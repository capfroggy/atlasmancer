# Atlasmancer Brand Foundation

Estado: inicial

## Nombre

Atlasmancer

## URL objetivo

`https://Atlasmancer.gt.tc`

## Taglines

English:

> Forge living worlds for tabletop campaigns.

Espanol:

> Crea mundos vivos para tus campanas de rol.

## Descripcion corta

English:

Atlasmancer is a free, open-source worldbuilding tool for tabletop campaigns.
Create continents, regions, countries, settlements, NPCs, quests, factions,
dungeons, printable maps, and player-safe exports in one place.

Espanol:

Atlasmancer es una herramienta gratuita y open-source para crear mundos de
campana. Crea continentes, regiones, paises, asentamientos, NPCs, misiones,
facciones, mazmorras, mapas imprimibles y exports seguros para jugadores en un
solo lugar.

## Personalidad

- Fantasiosa, pero practica.
- Creativa, pero organizada.
- Amigable para DMs nuevos.
- Profunda para worldbuilders obsesivos.
- Abierta, comunitaria y modificable.

## Que debe evitar

- No sonar corporativo.
- No depender de la marca D&D.
- No prometer IA magica que hace todo sola.
- No ser solo un generador de imagenes.
- No esconder los datos del usuario en un backend cerrado.

## Primer estilo visual

Direccion inicial:

- Parchment/fantasy atlas para exports imprimibles.
- Interfaz web clara y utilitaria para edicion.
- Colores ricos, no solo marron/pergamino.
- Mapas legibles antes que decorativos.
- Modo jugador y modo DM claramente separados.
